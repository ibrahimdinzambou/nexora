# IPTV SaaS Frontend

Frontend statique de l'application IPTV SaaS.

Ce dossier contient les pages HTML, CSS, JavaScript et assets initialement servis par Spring Boot depuis `src/main/resources/static`.

Les appels API utilisent les routes `/api/...`; en deploiement separe, servez ce frontend derriere le meme domaine que l'API ou configurez un proxy vers l'API.
